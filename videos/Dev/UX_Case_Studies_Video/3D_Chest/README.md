## Video Specifications:

- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (Full HD) or 3840x2160 (4K)
- Frame Rate: 30fps or 24fps
- Duration: Keep main videos under 30 seconds for web performance
- Compression: Balance quality and file size (aim for under 10MB per video)

## Rendering Tips:

### For Turntable Animation:
1. Set up camera on a circular path around the chest
2. Use 360 frames for smooth rotation (1 frame per degree)
3. Enable motion blur for professional look
4. Render at 30fps
5. Use consistent lighting throughout

### Blender Export Settings:
- Container: MPEG-4
- Video Codec: H.264
- Output Quality: High quality
- Encoding Speed: Good
- Keyframe interval: 10

### Post-Processing:
- Consider adding subtle color grading
- Add fade in/out transitions (1-2 seconds)
- Keep file size optimized for web (use HandBrake or similar if needed)
