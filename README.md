# 3D Chest Project Assets

This folder contains images for the 3D Chest case study.

## Recommended Images to Add:

1. **hero-render.jpg/png** - Main hero image showing the final rendered chest (high quality, dramatic lighting)

2. **wireframe.jpg/png** - Wireframe view showing the topology and edge flow

3. **materials-nodes.jpg/png** - Screenshot of Blender's shader nodes showing material setup

4. **lighting-setup.jpg/png** - Screenshot showing the lighting rig and setup

5. **detail-shot-1.jpg/png** - Close-up of wood grain and texture details

6. **detail-shot-2.jpg/png** - Close-up of metal hardware (hinges, lock, bands)

7. **uv-layout.jpg/png** - UV unwrapping layout

8. **process-shots/** (optional subfolder) - Multiple work-in-progress screenshots showing the modeling stages

## Image Specifications:

- Format: JPG or PNG
- Resolution: Minimum 1920x1080 for main images
- Color: sRGB color space
- Compression: High quality (minimal compression for hero images)

## Tips for Captures:

- Use high sample count renders (256+ samples) for final images
- Enable denoising for clean results
- Consider using HDRI lighting for realistic reflections
- Capture multiple angles to showcase different details
- Use Blender's viewport screenshot for wireframe/materials views
